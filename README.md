**Community Connection Challenge - The Local Services Trust Problem**

Develop an intelligent healthcare assistance system that helps users understand and categorize their
symptoms, providing a preliminary, non-diagnostic assessment and guiding them toward appropriate medical
experts or hospitals. The system should analyze user-reported symptoms, identify potential concern areas, and
offer tailored suggestions such as nearby specialized hospitals, emergency indicators, and recommended
departments or specialists.
The goal is to support users with early symptom awareness, reduce confusion during medical decision-making,
and streamline access to suitable healthcare servicesâ€”while clearly emphasizing that it does not replace
professional medical diagnosis.

## ğŸš€ Getting Started

### Prerequisites
- Node.js
- npm or yarn
- Python

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ParthShikhare19/SymptoGuide
   cd SymptoGuide
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:8080`

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```
